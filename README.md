# Institutional Psychology Code

Open computational scaffolding for the Institutional Psychology knowledge series.

This repository supports article-level examples for rules, norms, legitimacy, authority, trust, compliance, institutional incentives, collective action, information flow, institutional memory, institutional learning, governance systems, path dependence, reform, crisis, and institutional resilience.

## Structure

Each article folder contains:

- `python/`
- `r/`
- `julia/`
- `cpp/`
- `fortran/`
- `c/`
- `rust/`
- `go/`
- `sql/`
- `notebooks/`
- `docs/`
- `data/raw/`
- `data/processed/`
- `outputs/`
