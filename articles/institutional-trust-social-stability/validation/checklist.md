# Institutional Trust Validation Checklist

Before adapting this scaffold to a real case, review:

- [ ] Is the institution or trust domain clearly defined?
- [ ] Is trust separated from surface compliance?
- [ ] Is consistency assessed?
- [ ] Is competence assessed?
- [ ] Is fairness assessed?
- [ ] Is transparency assessed?
- [ ] Is accountability assessed?
- [ ] Is integrity assessed?
- [ ] Is recognition and voice assessed?
- [ ] Is repair capacity assessed?
- [ ] Is legitimacy assessed separately from trust?
- [ ] Is voluntary compliance assessed separately from coercive compliance?
- [ ] Is administrative burden assessed?
- [ ] Is arbitrariness pressure assessed?
- [ ] Is visible violation pressure assessed?
- [ ] Is trust disaggregated across groups?
- [ ] Are historical harms and institutional memory included?
- [ ] Are affected communities visible in the analysis?
- [ ] Are qualitative sources included?
- [ ] Are complaint and appeal pathways reviewed?
- [ ] Is uncertainty documented?
- [ ] Are results framed as interpretive rather than deterministic?
- [ ] Is the analysis free from surveillance, trust scoring, or individual assessment uses?
