# Conceptual Model

A semi-formal model of institutional trust:

```text
IT = f(CS, CP, FR, TR, AC, IN, RG, RP, EX, VV, AB)
```

Where:

| Symbol | Meaning |
|---|---|
| IT | Institutional Trust |
| CS | Consistency |
| CP | Competence |
| FR | Fairness |
| TR | Transparency |
| AC | Accountability |
| IN | Integrity |
| RG | Recognition and Voice |
| RP | Repair Capacity |
| EX | Arbitrariness Pressure |
| VV | Visible Violation Pressure |
| AB | Administrative Burden |

A simplified trust score:

```text
IT = β1(CS) + β2(CP) + β3(FR) + β4(TR) + β5(AC)
   + β6(IN) + β7(RG) + β8(RP)
   − β9(EX) − β10(VV) − β11(AB)
```

A social-stability equation:

```text
SS = λ1(IT) + λ2(LG) + λ3(VC) + λ4(CO) + λ5(LR)
   − λ6(AR) − λ7(FRG)
```

Where:

- `SS` = social stability
- `LG` = legitimacy
- `VC` = voluntary compliance
- `CO` = cooperation capacity
- `LR` = learning and repair
- `AR` = arbitrariness pressure
- `FRG` = fragmentation pressure

A trust-repair asymmetry equation:

```text
T(t+1) = T(t) + λR(t) − ωH(t)
```

Where `H(t)` is visible harm or violation and `R(t)` is credible repair. In many trust systems, `ω > λ`, meaning harm destroys trust faster than routine repair rebuilds it.

## Interpretation

- Trust is not public-relations sentiment.
- Trust is an earned behavioral expectation.
- Trust is easier to damage than rebuild.
- Trust is distributed unevenly across groups and histories.
- Stable order is not always trust-based; it may be coercive, procedural, exclusionary, or fragile.
