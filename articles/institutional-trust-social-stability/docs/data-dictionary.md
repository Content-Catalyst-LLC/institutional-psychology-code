# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic institutional trust unit identifier |
| period | integer | Simulated trust period |
| consistency | numeric | Predictable rule application and institutional behavior |
| competence | numeric | Capacity to perform core institutional functions |
| fairness | numeric | Procedural and distributive fairness perception |
| transparency | numeric | Intelligibility and visibility of institutional reasoning |
| accountability | numeric | Credible correction of errors, abuses, failures, and harms |
| integrity | numeric | Alignment between institutional claims and conduct |
| recognition_voice | numeric | Affected people treated as legitimate participants and knowers |
| repair_capacity | numeric | Acknowledgment, correction, restitution, memory, and institutional redesign capacity |
| legitimacy | numeric | Perceived credibility and rightfulness of institutional authority |
| voluntary_compliance | numeric | Compliance rooted in trust and legitimacy rather than coercion |
| cooperation_capacity | numeric | Capacity for coordinated behavior across groups and systems |
| learning_repair | numeric | Ability to convert failure into learning and redesign |
| arbitrariness_pressure | numeric | Inconsistency, bias, unpredictable treatment, or arbitrary conduct pressure |
| visible_violation_pressure | numeric | Hypocrisy, corruption, abuse, or public institutional breach pressure |
| fragmentation_pressure | numeric | Social, political, organizational, or community fragmentation pressure |
| administrative_burden | numeric | Proof, waiting, complexity, documentation, stigma, and procedural load |
| trust_score | numeric | Composite synthetic institutional trust score |
| social_stability | numeric | Composite synthetic social-stability score |
| high_trust | binary | High-trust threshold indicator |
| high_stability | binary | High-stability threshold indicator |
| fragile_trust_environment | binary | High apparent trust with weak fairness and weak accountability |
| high_distrust_pressure | binary | Strong arbitrariness and violation pressure with weak repair capacity |
