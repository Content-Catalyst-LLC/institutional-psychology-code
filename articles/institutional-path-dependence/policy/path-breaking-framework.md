# Path-Breaking Reform Framework

## Reform quality criteria

A path-breaking reform should be evaluated across multiple criteria:

1. **Historical diagnosis**  
   Does it identify the sequence that produced lock-in?

2. **Mechanism clarity**  
   Does it specify which reinforcing mechanisms must change?

3. **Credible alternative**  
   Does it offer a workable replacement path?

4. **Implementation capacity**  
   Are resources, authority, and timelines aligned with reform goals?

5. **Legitimacy reconstruction**  
   Does it build trust in the new path?

6. **Distributional justice**  
   Does it reduce unequal burdens embedded in the old path?

7. **Complementarity alignment**  
   Does it address adjacent institutions that sustain the path?

8. **Learning capacity**  
   Does it improve feedback, memory, and future adaptation?

## Reform outcome typology

| Outcome | Description |
|---|---|
| Surface reform | Language changes but the path remains intact |
| Procedural reform | Processes change but deeper incentives remain |
| Path adjustment | The system changes direction modestly without full transformation |
| Path breaking | Reinforcing mechanisms are disrupted and a new trajectory begins |
| Path substitution | A new institutional arrangement replaces the old path |
| Regressive lock-in | Reform creates a new but less accountable path |
