# Path Dependence Validation Checklist

Before adapting this scaffold to a real case, review:

- [ ] Are initial conditions clearly identified?
- [ ] Is there evidence of reinforcement over time?
- [ ] Are switching costs documented?
- [ ] Are coordination effects documented?
- [ ] Is legitimacy measured across groups, not only in aggregate?
- [ ] Is complementarity with adjacent institutions documented?
- [ ] Are marginalized communities visible in the analysis?
- [ ] Does the model separate persistence from effectiveness?
- [ ] Does it separate lock-in from legitimate continuity?
- [ ] Are qualitative sources included?
- [ ] Are historical sources included?
- [ ] Are assumptions documented?
- [ ] Is uncertainty documented?
- [ ] Are results framed as interpretive rather than deterministic?
- [ ] Is the analysis free from workplace surveillance or individual assessment uses?
