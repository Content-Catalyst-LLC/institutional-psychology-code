# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| institution_id | integer | Synthetic institutional identifier |
| period | integer | Simulated period |
| initial_conditions | numeric | Strength of early path-shaping conditions |
| behavioral_reinforcement | numeric | Repeated behavior reproducing the path |
| feedback_strength | numeric | Strength of rewards for path-aligned behavior |
| increasing_returns | numeric | Benefits that grow as adoption increases |
| coordination_effects | numeric | Benefits from shared alignment |
| learning_effects | numeric | Familiarity and competence inside the existing path |
| legitimacy | numeric | Perceived rightfulness and normality of the path |
| switching_costs | numeric | Cost of moving away from the existing path |
| complementarity | numeric | Interdependence with surrounding institutional arrangements |
| disruption_pressure | numeric | Crisis, reform, or path-breaking pressure |
| reform_capacity | numeric | Capacity to build and implement credible alternatives |
| distributional_burden | numeric | Unequal harm or constraint created by the path |
| path_strength | numeric | Dynamic strength of the institutional path |
| path_dependence_score | numeric | Composite synthetic path-dependence score |
| stay_probability | numeric | Probability of staying on the current path |
| lock_in | binary | Path-dependence threshold indicator |
| strong_lock_in | binary | Strong lock-in threshold indicator |
| high_burden_lock_in | binary | Lock-in combined with high distributional burden |
