# Conceptual Model

A semi-formal model of institutional path dependence:

```text
PD = f(IC, BR, FB, IR, CO, LE, LG, SC, CP, DP, RC, DB)
```

Where:

| Symbol | Meaning |
|---|---|
| PD | Path Dependence |
| IC | Initial Conditions |
| BR | Behavioral Reinforcement |
| FB | Feedback Strength |
| IR | Increasing Returns |
| CO | Coordination Effects |
| LE | Learning Effects |
| LG | Legitimacy |
| SC | Switching Costs |
| CP | Complementarity |
| DP | Disruption Pressure |
| RC | Reform Capacity |
| DB | Distributional Burden |

A simplified path-dependence score:

```text
PD = α1(IC) + α2(BR) + α3(FB) + α4(IR) + α5(CO) + α6(LE)
   + α7(LG) + α8(SC) + α9(CP) − α10(DP) − α11(RC)
```

A simplified lock-in probability model:

```text
P(LockIn_t) = σ(β0 + β1IR_t + β2CO_t + β3LG_t + β4SC_t
                  + β5CP_t − β6DP_t − β7RC_t)
```

## Interpretation

- Early conditions establish the path.
- Reinforcing mechanisms stabilize it.
- Legitimacy converts familiarity into normative authority.
- Switching costs make deviation costly.
- Complementarity binds the path to adjacent institutions.
- Reform capacity and disruption pressure can weaken lock-in.
- Distributional burden asks who pays the cost of continuity.
