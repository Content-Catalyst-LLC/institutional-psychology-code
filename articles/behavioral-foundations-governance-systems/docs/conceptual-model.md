# Conceptual Model

A semi-formal model of behavioral governance effectiveness:

```text
GE = f(IN, LG, NO, CG, TR, CO, EN, AL, FA, BB, HY, PA)
```

Where:

| Symbol | Meaning |
|---|---|
| GE | Governance Effectiveness |
| IN | Incentive Alignment |
| LG | Legitimacy |
| NO | Norm Support |
| CG | Cognitive Interpretability |
| TR | Trust |
| CO | Coordination Quality |
| EN | Enforcement Credibility |
| AL | Adaptive Learning Capacity |
| FA | Perceived Fairness |
| BB | Behavioral Burden |
| HY | Hypocrisy Visibility |
| PA | Power Asymmetry |

A simplified governance score:

```text
GE = β1(IN) + β2(LG) + β3(NO) + β4(CG) + β5(TR)
   + β6(CO) + β7(EN) + β8(AL) + β9(FA)
   − β10(BB) − β11(HY) − β12(PA)
```

A simplified compliance probability model:

```text
P(Comply_i) = σ(α0 + α1L_i + α2T_i + α3F_i + α4N_i
                   + α5E_i + α6C_i − α7B_i − α8H_i − α9P_i)
```

Where:

- `L_i` = legitimacy
- `T_i` = trust
- `F_i` = perceived fairness
- `N_i` = norm compatibility
- `E_i` = enforcement credibility
- `C_i` = cognitive interpretability
- `B_i` = behavioral burden
- `H_i` = hypocrisy visibility
- `P_i` = power asymmetry exposure

## Interpretation

- Governance effectiveness depends on behaviorally workable institutional design.
- Enforcement is strongest when supported by legitimacy, fairness, and trust.
- High compliance may still indicate fragile governance if burden, hypocrisy, or power asymmetry are high.
- Governance systems should be evaluated by performance, legitimacy, burden, voice, and learning capacity.
