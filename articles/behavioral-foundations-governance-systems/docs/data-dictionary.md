# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic governance unit identifier |
| agent_id | integer | Synthetic actor identifier |
| period | integer | Simulated time period |
| incentive_alignment | numeric | Alignment between institutional incentives and stated governance goals |
| legitimacy | numeric | Perceived rightfulness, procedural fairness, and authority acceptance |
| norm_support | numeric | Informal expectation support for formal governance |
| cognitive_interpretability | numeric | Ease of understanding and acting on rules or procedures |
| trust | numeric | Confidence in institutional competence, honesty, and fair administration |
| coordination_quality | numeric | Degree of alignment across roles, timing, actors, and expectations |
| enforcement_credibility | numeric | Consistency, proportionality, and credibility of enforcement |
| adaptive_learning | numeric | Capacity to detect error, interpret feedback, and revise practice |
| perceived_fairness | numeric | Perceived fairness of governance procedures and outcomes |
| behavioral_burden | numeric | Learning, compliance, psychological, documentation, and appeal costs |
| hypocrisy_visibility | numeric | Visibility of powerful actors violating rules without accountability |
| power_asymmetry | numeric | Unequal exposure to monitoring, sanction, burden, or lack of voice |
| governance_effectiveness | numeric | Synthetic governance-effectiveness score |
| high_governance | binary | High governance threshold indicator |
| fragile_governance | binary | High governance with low trust |
| high_burden_governance | binary | High governance with high burden and weak fairness |
