# Reform Evaluation Framework

## Reform quality criteria

A crisis-driven reform should be evaluated across multiple criteria:

1. **Substantive correction**  
   Does it address root causes or only visible symptoms?

2. **Legitimacy reconstruction**  
   Does it restore trust through transparency, accountability, and public justification?

3. **Implementation capacity**  
   Are resources, authority, staffing, and timelines aligned with reform goals?

4. **Distributional justice**  
   Does it reduce unequal exposure to harm?

5. **Accountability**  
   Are decision rights, review processes, and remedies clear?

6. **Learning capacity**  
   Does the reform improve feedback, memory, and adaptation?

7. **Capture resistance**  
   Does it protect reform from insider self-protection or elite capture?

8. **Durability**  
   Can the reform survive beyond the initial crisis window?

## Reform outcome typology

| Outcome | Description |
|---|---|
| Symbolic reform | Language changes but core incentives remain unchanged |
| Procedural reform | Rules and processes change but authority remains similar |
| Structural reform | Authority, resources, and accountability are redesigned |
| Transformational reform | Institutional logic, legitimacy, and behavior are reconstituted |
| Captured reform | Reform is redirected toward powerful interests |
| Regressive reform | Crisis enables less accountable or more exclusionary institutions |
