# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| case_id | integer | Synthetic case identifier |
| institution_id | integer | Synthetic institutional identifier |
| period | integer | Simulated period |
| crisis_severity | numeric | Magnitude and visibility of disruption |
| feedback_breakdown | numeric | Degree of institutional signal failure |
| legitimacy_failure | numeric | Loss of perceived authority, fairness, and credibility |
| legitimacy | numeric | Remaining legitimacy in dynamic simulation |
| adaptive_capacity | numeric | Ability to revise routines under uncertainty |
| reform_window | numeric | Openness of conditions for reform |
| coalition_strength | numeric | Organized capacity for reform |
| coordination_quality | numeric | Cross-unit or cross-level alignment |
| learning_rate | numeric | Ability to convert feedback into institutional redesign |
| governance_alignment | numeric | Fit between reform goals, authority, and implementation capacity |
| power_concentration | numeric | Concentration of authority during crisis and reform |
| capture_risk | numeric | Risk that reform is redirected toward insider or elite interests |
| distributional_attention | numeric | Attention to unequal harm and marginalized communities |
| transformation_score | numeric | Composite synthetic reform-depth score |
| major_reform | binary | Meaningful reform threshold indicator |
| deep_transformation | binary | Deep transformation threshold indicator |
| high_reform_likelihood | binary | Dynamic reform-probability threshold indicator |
