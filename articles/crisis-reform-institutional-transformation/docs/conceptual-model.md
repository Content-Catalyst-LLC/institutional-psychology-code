# Conceptual Model

A semi-formal model of crisis-driven transformation:

```text
IT = f(CS, FB, LF, AC, RW, PC, CO, LR, GA, RC, CR, DA)
```

Where:

| Symbol | Meaning |
|---|---|
| IT | Institutional Transformation |
| CS | Crisis Severity |
| FB | Feedback Breakdown |
| LF | Legitimacy Failure |
| AC | Adaptive Capacity |
| RW | Reform Window Openness |
| PC | Power Configuration |
| CO | Coordination Quality |
| LR | Learning Rate |
| GA | Governance Alignment |
| RC | Reform Coalition Strength |
| CR | Capture Risk |
| DA | Distributional Attention |

A simplified transformation score:

```text
IT = β1(CS) + β2(FB) + β3(LF) + β4(AC) + β5(RW) + β6(RC)
   + β7(LR × GA) + β8(CO) + β9(DA) − β10(CR) ± β11(PC)
```

A simplified reform probability model:

```text
P(Reform_t) = σ(γ0 + γ1CS_t + γ2LF_t + γ3RW_t + γ4RC_t
                + γ5AC_t + γ6DA_t − γ7CR_t)
```

## Interpretation

- Crisis severity creates pressure.
- Legitimacy failure weakens inherited authority.
- Coalition strength converts pressure into action.
- Adaptive capacity makes reform implementable.
- Capture risk can distort reform even when reform appears likely.
- Distributional attention connects reform to justice and legitimacy.
