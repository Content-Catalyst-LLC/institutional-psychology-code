# Validation Plan

A real crisis and reform study should validate claims across several layers.

## 1. Construct validity

Confirm that indicators actually measure the intended concepts.

Examples:

- Does "legitimacy failure" reflect public trust, staff trust, procedural fairness, or all three?
- Does "reform adoption" capture paper reform or implemented change?
- Does "coalition strength" include affected communities or only formal elites?

## 2. Historical validation

Compare findings against archival sources, public inquiries, legislative records, investigative reports, administrative histories, and long-term institutional trajectories.

## 3. Process-tracing validation

Ask whether the proposed causal sequence is plausible:

```text
crisis → legitimacy failure → reform window → coalition formation → institutional change
```

## 4. Sensitivity analysis

Test whether conclusions change when:

- weights are altered
- capture risk is increased
- legitimacy thresholds are changed
- reform-window assumptions vary
- distributional attention is included or excluded

## 5. Distributional validation

Check whether aggregate reform success hides unequal harm or unequal recovery.

## 6. Participatory validation

Invite affected communities, frontline staff, civil society actors, and domain experts to review interpretation.
