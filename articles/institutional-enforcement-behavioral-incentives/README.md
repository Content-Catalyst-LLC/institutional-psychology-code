# Institutional Enforcement and Behavioral Incentives

This advanced companion repository supports the article **Institutional Enforcement and Behavioral Incentives** in the Institutional Psychology knowledge series.

The repository provides a research-grade, synthetic-data scaffold for studying how monitoring, sanctions, rewards, audits, corrective intervention, detection visibility, information quality, legitimacy, incentive alignment, adaptive learning, accountability reach, compliance burden, selective enforcement, defensive compliance, hypocrisy visibility, and evasion pressure shape behavior in institutional systems.

## Research purpose

This repository is designed for:

- synthetic-data research
- enforcement-effectiveness modeling
- monitoring and information-quality analysis
- sanction-legitimacy interaction analysis
- incentive-alignment diagnostics
- adaptive-evasion and gaming review
- compliance-burden analysis
- selective-enforcement review
- fragile enforcement diagnostics
- high-burden enforcement review
- accountability-reach analysis
- institutional learning and correction modeling
- public administration and regulatory behavior examples
- organizational compliance and internal-control examples
- platform governance and automated-enforcement examples
- reproducible multi-language workflows
- teaching and public-interest methods demonstration

It is not designed for automated decision-making about real people, employees, citizens, patients, students, applicants, agencies, firms, communities, or institutions.

## Core research question

When does enforcement become legitimate, learning-oriented accountability rather than symbolic control, fear-driven compliance, selective discipline, defensive documentation, or adaptive evasion?

## Conceptual frame

The scaffold models enforcement effectiveness as a function of:

- monitoring quality
- legitimacy
- incentive alignment
- sanction credibility
- information quality
- adaptive learning
- accountability reach
- compliance burden
- selective enforcement
- evasion pressure
- hypocrisy visibility
- defensive compliance
- disclosure safety
- correction quality
- burden justice

## Main variables

| Variable | Meaning |
|---|---|
| monitoring_quality | Coverage, accuracy, timeliness, fairness, and interpretability of monitoring |
| legitimacy | Perceived fairness, rightfulness, and institutional credibility of enforcement |
| incentive_alignment | Degree to which enforcement supports institutional purpose rather than gaming |
| sanction_credibility | Consistency, proportionality, and meaningfulness of sanctions |
| information_quality | Accuracy, completeness, auditability, and usefulness of enforcement information |
| adaptive_learning | Capacity to revise rules, monitoring, sanctions, and procedures from feedback |
| accountability_reach | Degree to which enforcement reaches decision-makers with authority |
| compliance_burden | Cost, complexity, documentation, time, and psychological burden of compliance |
| selective_enforcement | Unequal monitoring, sanction, burden, flexibility, or consequence |
| evasion_pressure | Incentive or opportunity to avoid, game, delay, or conceal enforcement |
| hypocrisy_visibility | Visibility of powerful actors avoiding accountability |
| defensive_compliance | Compliance behavior focused on blame avoidance rather than substantive correction |
| enforcement_effectiveness | Composite synthetic enforcement-effectiveness score |
| high_compliance_quality | Binary threshold indicator for strong compliance quality |
| fragile_enforcement | High apparent effectiveness with low legitimacy |
| high_burden_enforcement | High apparent effectiveness with high burden and selective enforcement |

## Responsible-use note

All data is synthetic. The repository is intended for conceptual modeling, public-interest research, education, and transparent methods demonstration.

Do not use this repository for:

- employee screening
- employment selection
- hiring
- promotion
- compensation
- discipline
- termination
- workplace surveillance
- individual performance management
- psychological assessment
- predictive policing
- automated eligibility determination
- coercive public administration
- ranking real communities, agencies, firms, workers, or institutions without context
- suppressing dissent under the language of enforcement or compliance
- labeling inability, exclusion, ambiguity, fear, or distrust as evasion
- legitimacy laundering for already-decided compliance or governance agendas

Any real-world adaptation must include legal review, ethical review, qualitative evidence, historical analysis, stakeholder participation, burden audits, affected-community voice, and attention to marginalized communities.

## Suggested workflow

1. Read `docs/research-design.md`.
2. Review `ethics/responsible-use-boundaries.md`.
3. Generate synthetic enforcement data with `python/generate_institutional_enforcement_data.py`.
4. Run dynamic enforcement simulation with `python/simulate_enforcement_dynamics.py`.
5. Run fragile enforcement diagnostics with `python/fragile_enforcement_review.py`.
6. Run compliance burden review with `python/compliance_burden_review.py`.
7. Run evasion sensitivity analysis with `python/evasion_pressure_sensitivity.py`.
8. Analyze enforcement outcomes in R with `r/institutional_enforcement_modeling.R`.
9. Create SQLite schema with `sql/schema.sql`.
10. Review `validation/validation-plan.md` and `validation/checklist.md`.
11. Use `docs/github-embed-wordpress.html` for the WordPress GitHub block.
