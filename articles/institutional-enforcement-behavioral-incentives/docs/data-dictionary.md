# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic enforcement unit identifier |
| period | integer | Simulated time period |
| monitoring_quality | numeric | Coverage, accuracy, timeliness, fairness, and interpretability of monitoring |
| legitimacy | numeric | Perceived fairness, rightfulness, and credibility of enforcement |
| incentive_alignment | numeric | Alignment between enforcement incentives and institutional purpose |
| sanction_credibility | numeric | Consistency, proportionality, and meaningfulness of sanctions |
| information_quality | numeric | Accuracy, completeness, auditability, and usability of enforcement information |
| adaptive_learning | numeric | Capacity to detect adaptation, revise procedures, and update practice |
| accountability_reach | numeric | Degree to which enforcement reaches decision-makers with authority |
| compliance_burden | numeric | Cost, complexity, documentation, time, and psychological burden of compliance |
| selective_enforcement | numeric | Uneven monitoring, sanction, burden, flexibility, or consequence |
| evasion_pressure | numeric | Incentive or opportunity to avoid, game, delay, or conceal enforcement |
| hypocrisy_visibility | numeric | Visibility of powerful actors avoiding accountability |
| defensive_compliance | numeric | Behavior focused on blame avoidance rather than substantive correction |
| enforcement_effectiveness | numeric | Composite synthetic enforcement-effectiveness score |
| high_compliance_quality | binary | High compliance-quality threshold indicator |
| fragile_enforcement | binary | High enforcement effectiveness with low legitimacy |
| high_burden_enforcement | binary | High enforcement effectiveness with high burden and selective enforcement |
