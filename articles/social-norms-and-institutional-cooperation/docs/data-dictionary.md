# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic institutional unit identifier |
| agent_id | integer | Synthetic actor identifier |
| period | integer | Simulated time period |
| descriptive_norm | numeric | Perceived frequency of norm-consistent behavior |
| injunctive_norm | numeric | Perceived approval or disapproval attached to behavior |
| trust | numeric | Belief that others will comply, reciprocate, or act predictably |
| legitimacy | numeric | Perceived fairness and rightfulness of the surrounding institution |
| sanction_intensity | numeric | Strength of expected informal sanction for deviation |
| transmission_strength | numeric | Strength of onboarding, socialization, learning, ritual, mentoring, or memory transmission |
| institutional_reinforcement | numeric | Degree to which formal systems reinforce the norm |
| norm_conflict | numeric | Conflict among competing normative expectations |
| hypocrisy_visibility | numeric | Visibility of powerful actors violating norms without consequence |
| unequal_enforcement | numeric | Unequal application of informal sanction |
| performative_compliance | numeric | Surface-level conformity without substantive norm enactment |
| distributional_attention | numeric | Attention to unequal norm-compliance burdens |
| cooperation_score | numeric | Synthetic norm-based cooperation score |
| high_norm_compliance | binary | High compliance threshold indicator |
| fragile_norm_environment | binary | High compliance with low trust |
| high_burden_norm_environment | binary | High compliance with unequal enforcement and weak distributional attention |
