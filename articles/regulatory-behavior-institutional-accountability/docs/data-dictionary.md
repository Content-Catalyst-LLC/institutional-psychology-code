# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic regulatory unit identifier |
| period | integer | Simulated time period |
| oversight_strength | numeric | Capacity, independence, authority, and intensity of oversight |
| legitimacy | numeric | Perceived fairness, rightfulness, and credibility of regulation |
| incentive_alignment | numeric | Alignment between regulatory incentives and public/institutional purpose |
| enforcement_credibility | numeric | Consistency, proportionality, and likelihood of enforcement |
| information_quality | numeric | Accuracy, completeness, auditability, and usability of regulatory information |
| adaptive_learning | numeric | Capacity to detect error, revise rules, and update regulatory practice |
| accountability_reach | numeric | Degree to which accountability reaches powerful actors |
| capture_pressure | numeric | Pressure from regulated interests, influence, information dependence, or proximity |
| regulatory_burden | numeric | Cost, complexity, documentation, time, and psychological burden of compliance |
| evasion_pressure | numeric | Incentive or opportunity to avoid, delay, game, or conceal compliance |
| hypocrisy_visibility | numeric | Visibility of powerful actors avoiding accountability |
| unequal_accountability | numeric | Uneven monitoring, burden, sanction, or consequence |
| accountability_effectiveness | numeric | Composite synthetic regulatory-accountability score |
| high_accountability | binary | High accountability threshold indicator |
| fragile_regulation | binary | High accountability with low legitimacy |
| high_burden_regulation | binary | High accountability with high burden and unequal accountability |
