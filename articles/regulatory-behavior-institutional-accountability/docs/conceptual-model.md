# Conceptual Model

A semi-formal model of regulatory accountability effectiveness:

```text
RA = f(OS, LG, IA, EC, IQ, AL, AR, RB, CP, EV, HY, UA)
```

Where:

| Symbol | Meaning |
|---|---|
| RA | Regulatory Accountability Effectiveness |
| OS | Oversight Strength |
| LG | Legitimacy |
| IA | Incentive Alignment |
| EC | Enforcement Credibility |
| IQ | Information Quality |
| AL | Adaptive Learning |
| AR | Accountability Reach |
| RB | Regulatory Burden |
| CP | Capture Pressure |
| EV | Evasion Pressure |
| HY | Hypocrisy Visibility |
| UA | Unequal Accountability |

A simplified accountability score:

```text
RA = β1(OS) + β2(LG) + β3(IA) + β4(EC) + β5(IQ)
   + β6(AL) + β7(AR)
   − β8(RB) − β9(CP) − β10(EV) − β11(HY) − β12(UA)
```

A simplified compliance probability model:

```text
P(Comply_i) = σ(α0 + α1L_i + α2F_i + α3O_i + α4T_i
                   + α5N_i + α6E_i − α7B_i − α8C_i)
```

Where:

- `L_i` = legitimacy
- `F_i` = perceived fairness
- `O_i` = oversight visibility
- `T_i` = trust in consistency
- `N_i` = norm support
- `E_i` = enforcement credibility
- `B_i` = compliance burden
- `C_i` = capture/evasion opportunity

## Interpretation

- Regulation is not merely rule presence; it is the behavioral organization of responsibility.
- Oversight requires information quality and independence.
- Enforcement requires legitimacy and proportionality to build durable accountability.
- High apparent accountability may be fragile when capture, burden, and unequal accountability are high.
