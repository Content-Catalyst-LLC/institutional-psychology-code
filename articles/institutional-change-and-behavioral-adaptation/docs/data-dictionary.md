# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| institution_id | integer | Synthetic institutional identifier |
| period | integer | Simulated time period |
| feedback_quality | numeric | Ability to detect, transmit, and use feedback |
| adaptive_capacity | numeric | Ability to revise routines under changing conditions |
| legitimacy | numeric | Perceived fairness, credibility, and rightfulness |
| incentive_alignment | numeric | Degree to which incentives support desired change |
| normative_support | numeric | Normative and role-based support for adaptation |
| governance_capacity | numeric | Capacity to interpret feedback and implement change |
| path_dependence | numeric | Institutional drag from inherited routines and constraints |
| behavioral_flexibility | numeric | Capacity of actors to adjust behavior |
| coordination_quality | numeric | Alignment across units, levels, and actors |
| environmental_change | numeric | External change pressure |
| distributional_attention | numeric | Attention to unequal burdens and affected voices |
| transition_burden | numeric | Cost, confusion, disruption, or overload caused by change |
| change_score | numeric | Composite synthetic institutional change score |
| adaptation_pressure | numeric | Dynamic pressure toward adaptation |
| institutional_strength | numeric | Simulated institutional capacity/continuity |
| successful_adaptation | binary | Adaptation threshold indicator |
| fragile_adaptation | binary | High-change but low-legitimacy indicator |
| high_transition_burden | binary | Transition burden threshold indicator |
