# Conceptual Model

A semi-formal model of institutional change and behavioral adaptation:

```text
IC = f(FB, AD, LG, IN, NO, GC, BF, CO, EC, PD, DA, TB)
```

Where:

| Symbol | Meaning |
|---|---|
| IC | Institutional Change |
| FB | Feedback Quality |
| AD | Adaptive Capacity |
| LG | Legitimacy |
| IN | Incentive Alignment |
| NO | Normative Support |
| GC | Governance Capacity |
| BF | Behavioral Flexibility |
| CO | Coordination Quality |
| EC | Environmental Change |
| PD | Path Dependence |
| DA | Distributional Attention |
| TB | Transition Burden |

A simplified institutional change score:

```text
IC = α1(FB) + α2(AD) + α3(LG) + α4(IN) + α5(NO) + α6(GC)
   + α7(BF) + α8(CO) + α9(EC) + α10(DA) − α11(PD) − α12(TB)
```

A simplified adaptation probability model:

```text
P(Adapt_t) = σ(β0 + β1FB_t + β2AD_t + β3LG_t + β4GC_t
                 + β5CO_t + β6DA_t − β7PD_t − β8TB_t)
```

## Interpretation

- Feedback signals create pressure for adjustment.
- Adaptive capacity and governance capacity turn feedback into action.
- Legitimacy supports cooperation during change.
- Path dependence and transition burden suppress adaptation.
- Distributional attention improves reform quality and legitimacy.
- Behavioral flexibility determines whether formal change becomes enacted practice.
