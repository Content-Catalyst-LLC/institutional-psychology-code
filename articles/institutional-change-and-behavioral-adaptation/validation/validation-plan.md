# Validation Plan

A real institutional change and behavioral adaptation study should validate claims across several layers.

## 1. Construct validity

Confirm that indicators actually measure the intended concepts.

Examples:

- Does "successful adaptation" measure real behavioral change or only formal reform?
- Does "legitimacy" reflect affected communities or only institutional insiders?
- Does "transition burden" capture who actually bears costs?

## 2. Behavioral validation

Compare formal reform claims with actual behavior:

- compliance patterns
- role changes
- workarounds
- frontline practices
- public use patterns
- resistance or exit behavior

## 3. Historical validation

Compare findings against archival sources, reform histories, administrative records, public reports, and long-term institutional trajectories.

## 4. Process-tracing validation

Ask whether the proposed sequence is plausible:

```text
feedback → governance interpretation → formal change → behavioral adaptation → legitimacy update
```

## 5. Sensitivity analysis

Test whether conclusions change when:

- weights are altered
- legitimacy thresholds change
- transition burden increases
- path dependence is stronger
- distributional attention is included or excluded

## 6. Distributional validation

Check whether aggregate adaptation hides unequal burdens across groups or places.

## 7. Participatory validation

Invite affected communities, frontline staff, civil society actors, and domain experts to review interpretation.
