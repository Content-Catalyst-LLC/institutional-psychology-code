# Authority and Legitimacy Validation Checklist

Before adapting this scaffold to a real case, review:

- [ ] Is the authority domain clearly defined?
- [ ] Is formal authority separated from earned legitimacy?
- [ ] Is obedience separated from voluntary recognition?
- [ ] Is formal authority clarity assessed?
- [ ] Is procedural legitimacy assessed?
- [ ] Is outcome legitimacy assessed?
- [ ] Is trust assessed?
- [ ] Is rule clarity assessed?
- [ ] Is social recognition assessed?
- [ ] Is accountability assessed?
- [ ] Is repair capacity assessed?
- [ ] Is fairness assessed?
- [ ] Is arbitrariness pressure assessed?
- [ ] Is visible inconsistency assessed?
- [ ] Is unequal burden assessed?
- [ ] Is opacity pressure assessed?
- [ ] Is enforcement coercion pressure assessed?
- [ ] Is legitimacy disaggregated across groups?
- [ ] Are historical harms and institutional memory included?
- [ ] Are affected communities visible in the analysis?
- [ ] Are qualitative sources included?
- [ ] Are complaint and appeal pathways reviewed?
- [ ] Is uncertainty documented?
- [ ] Are results framed as interpretive rather than deterministic?
- [ ] Is the analysis free from surveillance, legitimacy scoring, or individual assessment uses?
