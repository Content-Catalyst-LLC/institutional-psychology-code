# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic authority-legitimacy unit identifier |
| period | integer | Simulated legitimacy period |
| formal_authority_clarity | numeric | Clarity of offices, roles, decision rights, review pathways, and jurisdiction |
| procedural_legitimacy | numeric | Perceived fairness, respect, consistency, transparency, and reviewability |
| outcome_legitimacy | numeric | Perceived adequacy, competence, public value, or tolerability of outcomes |
| trust | numeric | Expectation that institutional authority will act consistently and in good faith |
| rule_clarity | numeric | Intelligibility of obligations, rights, classifications, and enforcement rules |
| social_recognition | numeric | Collective recognition of institutional rightfulness |
| accountability | numeric | Credible correction, review, appeal, responsibility, and consequences |
| repair_capacity | numeric | Acknowledgment, correction, restitution, memory, and redesign capacity |
| fairness | numeric | Procedural and distributive fairness across cases and groups |
| shared_norm_support | numeric | Social norm supporting rule recognition and voluntary compliance |
| arbitrariness_pressure | numeric | Inconsistency, opacity, arbitrary discretion, or abusive treatment pressure |
| visible_inconsistency | numeric | Visible gap between stated rule and actual practice |
| unequal_burden | numeric | Unequal proof, waiting, surveillance, sanction, or administrative burden |
| opacity_pressure | numeric | Difficulty understanding or contesting institutional decisions |
| enforcement_coercion_pressure | numeric | Reliance on threat, sanction, monitoring, surveillance, or force |
| authority_legitimacy_strength | numeric | Composite synthetic legitimacy-strength score |
| voluntary_compliance | numeric | Composite synthetic voluntary-compliance score |
| high_legitimacy | binary | High-legitimacy threshold indicator |
| high_voluntary_compliance | binary | High voluntary-compliance threshold indicator |
| fragile_legitimacy_environment | binary | High apparent legitimacy with weak procedure and weak trust |
| high_arbitrariness_environment | binary | Strong arbitrariness and visible inconsistency with weak repair capacity |
