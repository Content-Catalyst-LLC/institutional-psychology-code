# Conceptual Model

A semi-formal model of authority-legitimacy effectiveness:

```text
AL = f(FA, PL, OL, TR, CL, SR, AC, RP, FR, AP, VI, UB, OP)
```

Where:

| Symbol | Meaning |
|---|---|
| AL | Authority-Legitimacy Effectiveness |
| FA | Formal Authority Clarity |
| PL | Procedural Legitimacy |
| OL | Outcome Legitimacy |
| TR | Trust |
| CL | Rule Clarity |
| SR | Social Recognition |
| AC | Accountability |
| RP | Repair Capacity |
| FR | Fairness |
| AP | Arbitrariness Pressure |
| VI | Visible Inconsistency |
| UB | Unequal Burden |
| OP | Opacity Pressure |

A simplified legitimacy-strength score:

```text
AL = β1(FA) + β2(PL) + β3(OL) + β4(TR) + β5(CL)
   + β6(SR) + β7(AC) + β8(RP) + β9(FR)
   − β10(AP) − β11(VI) − β12(UB) − β13(OP)
```

A voluntary-compliance equation:

```text
VC = δ1(AL) + δ2(TR) + δ3(FR) + δ4(SN) + δ5(CL) + δ6(RP)
   − δ7(EC) − δ8(AP) − δ9(UB)
```

Where:

- `VC` = voluntary compliance
- `SN` = shared norm support
- `EC` = enforcement coercion pressure

A legitimacy-fragility equation:

```text
LF = λ1(AP) + λ2(VI) + λ3(UB) + λ4(HH) + λ5(OP)
   − λ6(PL) − λ7(AC) − λ8(RP) − λ9(SR)
```

Where:

- `LF` = legitimacy fragility
- `HH` = historical harm or institutional betrayal memory

## Interpretation

- Formal authority is not the same as legitimacy.
- Obedience is not the same as recognition.
- Stability is not the same as justice.
- Legitimacy is earned through procedure, performance, recognition, accountability, and repair.
- Legitimacy can be unevenly distributed across groups and histories.
