# Institutional Decision Quality Validation Checklist

Before adapting this scaffold to a real case, review:

- [ ] Is the decision system clearly defined?
- [ ] Are assumptions documented?
- [ ] Are evidence sources documented?
- [ ] Are excluded alternatives documented?
- [ ] Is bounded-rationality pressure assessed?
- [ ] Is organizational structure quality assessed?
- [ ] Is incentive alignment assessed?
- [ ] Is information-flow effectiveness assessed?
- [ ] Is legitimacy assessed?
- [ ] Is uncertainty management assessed?
- [ ] Is corrective capacity assessed?
- [ ] Is justice-sensitive voice assessed?
- [ ] Is memory quality assessed?
- [ ] Is feedback openness assessed?
- [ ] Is bias distortion assessed?
- [ ] Is power protection assessed?
- [ ] Is metric fixation assessed?
- [ ] Is siloing assessed?
- [ ] Is premature closure assessed?
- [ ] Are affected communities visible in the analysis?
- [ ] Are dissent and bad-news channels protected?
- [ ] Are qualitative sources included?
- [ ] Are historical sources included?
- [ ] Is uncertainty documented?
- [ ] Are results framed as interpretive rather than deterministic?
- [ ] Is the analysis free from workplace surveillance or individual assessment uses?
