# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic institutional decision unit identifier |
| period | integer | Simulated decision period |
| bounded_rationality_pressure | numeric | Cognitive, time, information, uncertainty, and complexity constraints |
| organizational_structure_quality | numeric | Quality of decision rights, review structures, escalation pathways, and role clarity |
| incentive_alignment | numeric | Degree to which rewarded behavior matches institutional purpose |
| information_flow_effectiveness | numeric | Ability to preserve signal and context across decision channels |
| legitimacy | numeric | Procedural fairness, transparency, reason-giving, contestability, and credibility |
| uncertainty_management | numeric | Scenario, sensitivity, monitoring, and staged-commitment capacity |
| corrective_capacity | numeric | Dissent, red-team review, premortems, audits, and revision capacity |
| justice_voice | numeric | Affected-community evidence, burden review, marginalized voice, and contestability |
| memory_quality | numeric | Preservation of assumptions, rationale, dissent, outcomes, and lessons |
| feedback_openness | numeric | Capacity to use negative evidence, complaints, weak signals, and outcomes |
| bias_distortion | numeric | Confirmation, overconfidence, anchoring, status quo, availability, and groupthink pressure |
| power_protection | numeric | Reputational, political, legal, budgetary, or authority-protective pressure |
| metric_fixation | numeric | Overreliance on visible indicators, targets, dashboards, rankings, or proxies |
| siloing | numeric | Fragmentation across units, professions, jurisdictions, or platforms |
| premature_closure | numeric | Early narrowing of alternatives, uncertainty, or dissent |
| decision_quality | numeric | Composite synthetic institutional decision-quality score |
| high_quality_decision | binary | High-quality decision environment threshold indicator |
| fragile_decision_environment | binary | High apparent quality with weak correction and weak information flow |
| high_distortion_environment | binary | High bias and power protection with weak feedback openness |
