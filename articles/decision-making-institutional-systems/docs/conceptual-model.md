# Conceptual Model

A semi-formal model of institutional decision quality:

```text
DQ = f(BR, OS, IA, IF, LG, UM, CC, JM, MR, FO, BD, PP, MF, SL, PC)
```

Where:

| Symbol | Meaning |
|---|---|
| DQ | Decision Quality |
| BR | Bounded-Rationality Pressure |
| OS | Organizational Structure Quality |
| IA | Incentive Alignment |
| IF | Information-Flow Effectiveness |
| LG | Legitimacy and Procedural Fairness |
| UM | Uncertainty-Management Capacity |
| CC | Corrective Capacity |
| JM | Justice-Sensitive Voice and Burden Evidence |
| MR | Memory Quality |
| FO | Feedback Openness |
| BD | Bias Distortion |
| PP | Power-Protective Pressure |
| MF | Metric Fixation |
| SL | Siloing |
| PC | Premature Closure |

A simplified decision-quality score:

```text
DQ = β1(OS) + β2(IA) + β3(IF) + β4(LG) + β5(UM)
   + β6(CC) + β7(JM) + β8(MR) + β9(FO)
   − β10(BR) − β11(BD) − β12(PP) − β13(MF) − β14(SL) − β15(PC)
```

A recursive decision-learning equation:

```text
DQ(t+1) = DQ(t) + αIF(t) + βCC(t) + γFO(t) + δMR(t) − λBD(t) − μPP(t)
```

Where:

- `IF(t)` = information-flow effectiveness
- `CC(t)` = corrective capacity
- `FO(t)` = feedback openness
- `MR(t)` = memory quality
- `BD(t)` = bias distortion
- `PP(t)` = power-protective pressure

## Interpretation

- Institutions do not make better decisions merely by collecting more data.
- Decision quality depends on how information, correction, legitimacy, incentives, justice, memory, and uncertainty management interact.
- Efficiency can accelerate error if institutions lack correction, feedback, and affected-community voice.
