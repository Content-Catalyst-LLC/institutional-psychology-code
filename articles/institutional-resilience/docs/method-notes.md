# Method Notes

## Composite index logic

The resilience index combines structural, behavioral, and governance dimensions:

- robustness
- adaptive capacity
- recovery capacity
- transformational capacity
- legitimacy
- trust
- feedback quality
- learning rate
- redundancy
- modularity
- coordination
- shock intensity

The illustrative model is:

```text
IR = α1(RB) + α2(AC) + α3(RC) + α4(TC) + α5(LG) + α6(TR)
   + α7(FB × LR) + α8(RD + MD + CO) − α9(EX)
```

## Why synthetic data?

Synthetic data allows transparent demonstration without exposing real people, real communities, confidential institutions, or sensitive administrative records.

## Why not automate decisions?

Institutional resilience analysis can be misused if converted into automated scoring systems for punitive management, surveillance, or legitimacy laundering. Any real-world adaptation should include participatory review, qualitative evidence, legal safeguards, and justice-sensitive interpretation.
