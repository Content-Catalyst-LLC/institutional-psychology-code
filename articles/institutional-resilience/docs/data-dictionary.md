# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| institution_id | integer | Synthetic institutional identifier |
| robustness | numeric | Ability to withstand immediate stress |
| adaptive_capacity | numeric | Ability to revise routines under changing conditions |
| recovery_capacity | numeric | Ability to restore core functions |
| transformational_capacity | numeric | Ability to reorganize when continuity is insufficient |
| legitimacy | numeric | Perceived rightfulness and justification of authority |
| trust | numeric | Confidence in competence, fairness, and institutional intelligibility |
| feedback_quality | numeric | Capacity to detect, transmit, and use feedback |
| learning_rate | numeric | Ability to convert feedback into improved routines |
| redundancy | numeric | Backup capacity and overlap |
| modularity | numeric | Ability to contain local failure |
| coordination | numeric | Cross-unit alignment and role clarity |
| shock_intensity | numeric | Strength of external/internal disruption |
| resilience_index | numeric | Composite synthetic resilience score |
| continuity_score | numeric | Estimated ability to preserve core functions |
| maintained_core_function | binary | Continuity threshold indicator |
