# Ethics and Scope Note

Institutional resilience is ethically complex. A system can be resilient while unjust. It can preserve order by shifting risk onto less protected groups. It can maintain continuity by exhausting frontline workers. It can protect dominant interests while presenting itself as stable.

This scaffold must not be used to rank real people, evaluate employees, automate governance decisions, or substitute for democratic accountability.

Any real-world adaptation should include:

- public-interest justification
- data minimization
- transparency
- participatory review
- attention to marginalized groups
- qualitative evidence
- legal review
- historical context
- explainability
- human accountability
