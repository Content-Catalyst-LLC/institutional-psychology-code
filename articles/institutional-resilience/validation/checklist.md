# Institutional Resilience Validation Checklist

Before adapting this scaffold to real-world institutional analysis, review:

- [ ] Are all variables clearly defined?
- [ ] Are indicators valid for the institution being studied?
- [ ] Are marginalized groups visible in the analysis?
- [ ] Does the model hide distributional burden?
- [ ] Were qualitative sources reviewed?
- [ ] Were historical cases compared?
- [ ] Were assumptions documented?
- [ ] Was sensitivity analysis performed?
- [ ] Were stakeholders consulted?
- [ ] Are results framed as interpretive, not deterministic?
- [ ] Is the workflow free from employee-screening or surveillance use?
