# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic institutional unit identifier |
| agent_id | integer | Synthetic actor identifier |
| period | integer | Simulated time period |
| trust | numeric | Belief that others will interpret and act predictably |
| information_quality | numeric | Quality of information available for alignment |
| communication_clarity | numeric | Clarity, consistency, and reach of institutional signals |
| focal_salience | numeric | Salience of the focal point, standard, or rule |
| authority_signal | numeric | Strength of authoritative coordination signal |
| norm_strength | numeric | Strength of informal expectations supporting alignment |
| learning_capacity | numeric | Capacity to detect and correct misalignment |
| uncertainty | numeric | Ambiguity about action, timing, or others' behavior |
| adaptation_burden | numeric | Switching, training, compliance, or implementation burden |
| competing_standards | numeric | Fragmentation across possible standards or equilibria |
| competing_authority | numeric | Ambiguity over whose signal should guide behavior |
| distributional_attention | numeric | Attention to unequal adaptation costs and affected voices |
| coordination_quality | numeric | Synthetic coordination-quality score |
| high_alignment | binary | High alignment threshold indicator |
| fragile_coordination | binary | High alignment with low trust |
| high_burden_coordination | binary | High alignment with high burden and weak distributional attention |
| coordination_rate | numeric | Simulated share of actors aligned in a period |
