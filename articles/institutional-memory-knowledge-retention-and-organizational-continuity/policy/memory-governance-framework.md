# Institutional Memory Governance Framework

## Memory quality criteria

An institutional memory system should be evaluated across multiple criteria:

1. **Retention**  
   Does the institution preserve records, rationale, decisions, metadata, and lessons?

2. **Tacit transfer**  
   Does experiential knowledge move across roles, generations, and transitions?

3. **Accessibility**  
   Can relevant actors find and use memory when decisions require it?

4. **Interpretive use**  
   Does memory shape current decisions, or does it remain inert?

5. **Revisability**  
   Can records, categories, assumptions, routines, and systems be updated responsibly?

6. **Technical continuity**  
   Do systems preserve data, provenance, schemas, rationale, and dependencies?

7. **Distributed integration**  
   Can memory move across departments, professions, systems, and communities?

8. **Memory justice**  
   Are affected-community memories, histories of harm, dissenting interpretations, and burden records preserved?

9. **Path-dependence review**  
   Are inherited assumptions periodically examined?

10. **Accountability**  
   Can official memory be contested, corrected, audited, and linked to responsibility?

## Outcome typology

| Outcome | Description |
|---|---|
| Usable memory | Preserved knowledge is accessible, contextualized, interpreted, and used |
| Inert archive | Records exist but do not inform decisions |
| Tacit dependency | Knowledge lives mostly in people and disappears through transition |
| Selective memory | Official memory preserves some narratives while excluding others |
| Technical amnesia | Systems preserve data without rationale, metadata, or design history |
| Path-dependent memory | Inherited memory narrows future possibility |
| Adaptive memory | Memory supports continuity while remaining revisable |
| Memory justice | Memory includes affected voices, harm records, and contestability |
