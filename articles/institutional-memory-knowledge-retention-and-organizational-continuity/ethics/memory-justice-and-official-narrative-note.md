# Memory Justice and Official Narrative Note

Institutional memory is not politically neutral.

A justice-sensitive memory analysis should ask:

- Whose experience becomes official memory?
- Whose evidence is treated as anecdotal?
- Whose harms are preserved?
- Whose harms are forgotten?
- Who controls classification?
- Who controls access?
- Who can correct or contest records?
- Does memory preserve accountability or protect reputation?
- Are dissenting interpretations retained?
- Are marginalized communities forced to carry memory because institutions failed to preserve it?

Institutional forgetting can be a form of harm when it forces people or communities to relive, restate, or re-prove experiences that institutions already had reason to know.
