# Institutional Memory Validation Checklist

Before adapting this scaffold to a real case, review:

- [ ] Is the memory system clearly defined?
- [ ] Are formal records documented?
- [ ] Are archives accessible and searchable?
- [ ] Are decision logs maintained?
- [ ] Is tacit knowledge transfer assessed?
- [ ] Is metadata quality assessed?
- [ ] Is technical continuity assessed?
- [ ] Is distributed integration assessed?
- [ ] Is path-dependence pressure assessed?
- [ ] Is loss fragmentation assessed?
- [ ] Is selective narration assessed?
- [ ] Is turnover pressure assessed?
- [ ] Is key-person dependency assessed?
- [ ] Are affected communities visible in memory?
- [ ] Are dissenting interpretations preserved?
- [ ] Are histories of harm and burden preserved?
- [ ] Are qualitative sources included?
- [ ] Are historical sources included?
- [ ] Are assumptions documented?
- [ ] Is uncertainty documented?
- [ ] Are results framed as interpretive rather than deterministic?
- [ ] Is the analysis free from workplace surveillance or individual assessment uses?
