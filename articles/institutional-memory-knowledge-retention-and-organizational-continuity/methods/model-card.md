# Model Card: Institutional Memory Scaffold

## Model purpose

This scaffold demonstrates how synthetic institutional data can be used to explore memory effectiveness, documented retention, tacit transfer, accessibility, interpretive use, revisability, technical continuity, metadata quality, distributed integration, memory justice, path dependence, loss fragmentation, selective narration, turnover pressure, and key-person dependency.

## Intended users

- researchers
- students
- policy analysts
- public-interest technologists
- institutional scholars
- governance researchers
- organizational researchers
- archival and records-governance researchers

## Out-of-scope users

- employers evaluating workers
- agencies ranking people or communities
- punitive knowledge-management systems
- automated decision systems
- surveillance systems

## Inputs

Synthetic values representing institutional memory conditions.

## Outputs

- memory effectiveness score
- high resilience memory indicator
- fragile memory indicator
- high path-dependence memory indicator
- dynamic memory simulation results
- knowledge-loss sensitivity analysis
- memory-fragility review

## Limitations

- synthetic data only
- no real causal validation
- no automatic policy recommendation
- no moral or legal responsibility assignment
- no classification of real actors or communities
- no substitute for legal, historical, archival, qualitative, or institutional expertise

## Ethical risks

- false precision
- treating official records as complete truth
- ignoring affected-community memory
- legitimizing selective institutional narratives
- using memory models to preserve power rather than accountability
