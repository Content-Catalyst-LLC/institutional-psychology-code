# Conceptual Model

A semi-formal model of institutional memory effectiveness:

```text
IM = f(DR, TK, AC, IU, RV, TC, MQ, DI, MJ, PD, LF, SN, TO, KI)
```

Where:

| Symbol | Meaning |
|---|---|
| IM | Institutional Memory Effectiveness |
| DR | Documented Retention |
| TK | Tacit Knowledge Continuity |
| AC | Accessibility and Retrieval Quality |
| IU | Interpretive Use |
| RV | Revisability |
| TC | Technical Continuity |
| MQ | Metadata Quality |
| DI | Distributed Integration |
| MJ | Memory Justice |
| PD | Path-Dependence Pressure |
| LF | Loss and Fragmentation Pressure |
| SN | Selective Narration |
| TO | Turnover Pressure |
| KI | Key-Person Dependency |

A simplified memory-effectiveness score:

```text
IM = β1(DR) + β2(TK) + β3(AC) + β4(IU) + β5(RV)
   + β6(TC) + β7(MQ) + β8(DI) + β9(MJ)
   − β10(PD) − β11(LF) − β12(SN) − β13(TO) − β14(KI)
```

A recursive memory equation:

```text
M(t+1) = M(t) + αD(t) + βT(t) + γR(t) − δL(t)
```

Where:

- `D(t)` = documented retention and archival quality
- `T(t)` = tacit transfer across people and roles
- `R(t)` = routine reinforcement and reuse in decisions
- `L(t)` = loss pressure through turnover, fragmentation, restructuring, or technical discontinuity

## Interpretation

- Institutions do not remember because information exists.
- Memory becomes useful when knowledge is preserved, transferred, accessible, contextualized, interpreted, contestable, revisable, and connected to action.
- High apparent memory can be fragile when it depends on key people, inaccessible archives, selective narration, outdated categories, or weak revisability.
