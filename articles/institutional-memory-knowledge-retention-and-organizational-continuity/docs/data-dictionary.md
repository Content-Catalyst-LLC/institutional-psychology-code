# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic institutional memory unit identifier |
| period | integer | Simulated time period |
| documented_retention | numeric | Quality and completeness of formal records, archives, and decision logs |
| tacit_transfer | numeric | Transfer of experiential knowledge across people, roles, and transitions |
| accessibility | numeric | Searchability, retrieval, permissions, discoverability, and practical usability |
| interpretive_use | numeric | Degree to which retained knowledge informs current decisions |
| revisability | numeric | Capacity to update categories, records, routines, assumptions, and systems |
| technical_continuity | numeric | Continuity of software, databases, schemas, infrastructure, and operational systems |
| metadata_quality | numeric | Context, provenance, authorship, versioning, rationale, and interpretive notes |
| distributed_integration | numeric | Capacity to connect dispersed memory across units, systems, people, and communities |
| memory_justice | numeric | Inclusion of affected-community memory, histories of harm, dissent, and burden records |
| path_dependence_pressure | numeric | Constraint from inherited categories, routines, precedents, and prior success narratives |
| loss_fragmentation | numeric | Pressure from siloed records, inaccessible archives, discontinuity, and fragmented systems |
| selective_narration | numeric | Official-memory bias toward reputation, hierarchy, or dominant interpretation |
| turnover_pressure | numeric | Knowledge-loss risk from churn, retirement, restructuring, outsourcing, or transition |
| key_person_dependency | numeric | Degree to which memory depends on a few people rather than durable systems |
| memory_effectiveness | numeric | Composite synthetic institutional memory-effectiveness score |
| high_resilience_memory | binary | High memory-effectiveness threshold indicator |
| fragile_memory | binary | High apparent memory with weak documentation and weak tacit transfer |
| high_path_dependence_memory | binary | High apparent memory with high path pressure and low revisability |
