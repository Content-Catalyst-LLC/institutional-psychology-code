# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic institutional decision unit identifier |
| period | integer | Simulated decision period |
| overconfidence | numeric | Excessive certainty, underestimated uncertainty, or overstated institutional capacity |
| anchoring_pressure | numeric | Reliance on initial estimates, historical baselines, inherited categories, or first frames |
| confirmation_pressure | numeric | Selective use of evidence that supports existing assumptions |
| conformity_pressure | numeric | Status, hierarchy, consensus, loyalty, or group-pressure effects |
| filtering_distortion | numeric | Softening, reframing, sanitizing, or delaying information as it moves |
| path_lock_in | numeric | Constraint from precedent, prior investments, historical success, or institutional memory |
| metric_tunnel_vision | numeric | Overreliance on visible indicators at the expense of context and lived evidence |
| power_protection | numeric | Pressure to protect reputation, authority, legitimacy, budgets, or dominant narratives |
| dissent_capacity | numeric | Strength of structured dissent, red-team review, premortems, and protected challenge |
| corrective_review | numeric | Quality of audits, independent review, retrospectives, and assumption testing |
| information_quality | numeric | Accuracy, relevance, context, source diversity, and timeliness of evidence |
| feedback_openness | numeric | Capacity to hear and act on negative evidence, complaints, near misses, and weak signals |
| psychological_safety | numeric | Safety to surface uncertainty, disagreement, error, and disconfirming evidence |
| justice_voice | numeric | Inclusion of affected-community evidence, burden audits, marginalized voice, and contestability |
| institutional_bias_pressure | numeric | Composite synthetic bias-pressure score |
| decision_quality | numeric | Composite synthetic institutional decision-quality score |
| high_resilience_decision | binary | High-quality decision environment threshold indicator |
| fragile_judgment | binary | High apparent decision quality with weak dissent and high filtering distortion |
| high_bias_environment | binary | High bias pressure with weak correction and weak feedback openness |
