# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic institutional unit identifier |
| agent_id | integer | Synthetic actor identifier |
| period | integer | Simulated time period |
| incentive_alignment | numeric | Alignment between private incentives and collective outcomes |
| trust | numeric | Belief that others will cooperate and institutions will administer contributions fairly |
| legitimacy | numeric | Perceived rightfulness, competence, and procedural fairness |
| norm_strength | numeric | Strength of cooperative norms and reciprocal expectations |
| enforcement_credibility | numeric | Credibility and fairness of monitoring and sanctions |
| communication_quality | numeric | Quality, clarity, and reach of information flows |
| coordination_quality | numeric | Quality of timing, role alignment, standards, and expectation convergence |
| perceived_fairness | numeric | Perceived fairness of contribution burdens and benefit distribution |
| free_riding_pressure | numeric | Incentive or opportunity to benefit without contributing |
| burden_inequality | numeric | Unequal distribution of contribution costs, risk, or hidden labor |
| hypocrisy_visibility | numeric | Visibility of powerful actors defecting without accountability |
| scale_complexity | numeric | Difficulty produced by scale, heterogeneity, anonymity, or jurisdictional fragmentation |
| cooperation_score | numeric | Synthetic cooperation-quality score |
| high_cooperation | binary | High cooperation threshold indicator |
| fragile_cooperation | binary | High cooperation with low trust |
| high_burden_cooperation | binary | High cooperation with high burden inequality and weak fairness |
