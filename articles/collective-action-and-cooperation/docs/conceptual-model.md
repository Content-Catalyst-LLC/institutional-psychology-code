# Conceptual Model

A semi-formal model of collective action capacity:

```text
CA = f(IN, TR, LG, NO, EN, CM, CO, FA, FR, BI, HC, SC)
```

Where:

| Symbol | Meaning |
|---|---|
| CA | Collective Action Capacity |
| IN | Incentive Alignment |
| TR | Trust |
| LG | Legitimacy |
| NO | Norm Strength |
| EN | Enforcement Credibility |
| CM | Communication Quality |
| CO | Coordination Quality |
| FA | Perceived Fairness |
| FR | Free-Riding Pressure |
| BI | Burden Inequality |
| HC | Hypocrisy Visibility |
| SC | Scale Complexity |

A simplified collective action score:

```text
CA = β1(IN) + β2(TR) + β3(LG) + β4(NO) + β5(EN)
   + β6(CM) + β7(CO) + β8(FA)
   − β9(FR) − β10(BI) − β11(HC) − β12(SC)
```

A simplified cooperation probability model:

```text
P(Cooperate_i) = σ(α0 + α1T_i + α2L_i + α3N_i + α4E_i
                      + α5C_i + α6F_i − α7R_i − α8B_i − α9H_i)
```

Where:

- `T_i` = trust
- `L_i` = legitimacy
- `N_i` = norm strength
- `E_i` = enforcement credibility
- `C_i` = communication quality
- `F_i` = perceived fairness
- `R_i` = free-riding pressure
- `B_i` = burden inequality exposure
- `H_i` = hypocrisy visibility

## Interpretation

- Cooperation is shaped by expectation, legitimacy, reciprocity, enforcement, and fairness.
- Free-riding pressure can weaken cooperation even when collective goals are valued.
- High cooperation does not prove fairness if burdens are hidden or unequally distributed.
- Collective action capacity and collective action legitimacy should be evaluated separately.
