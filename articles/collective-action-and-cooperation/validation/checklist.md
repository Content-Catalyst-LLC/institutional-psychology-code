# Collective Action Validation Checklist

Before adapting this scaffold to a real case, review:

- [ ] Is the collective action problem clearly defined?
- [ ] Is the shared good, risk, or outcome specified?
- [ ] Are contributors and beneficiaries identified?
- [ ] Is free-riding pressure assessed?
- [ ] Is inability distinguished from opportunistic free-riding?
- [ ] Is trust assessed?
- [ ] Is legitimacy assessed?
- [ ] Is enforcement credibility assessed?
- [ ] Is perceived fairness assessed?
- [ ] Is burden inequality assessed?
- [ ] Are hidden contributions considered?
- [ ] Are marginalized communities visible in the analysis?
- [ ] Are qualitative sources included?
- [ ] Are historical sources included?
- [ ] Are assumptions documented?
- [ ] Is uncertainty documented?
- [ ] Are results framed as interpretive rather than deterministic?
- [ ] Is the analysis free from workplace surveillance or individual assessment uses?
