# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic institutional unit identifier |
| agent_id | integer | Synthetic agent identifier |
| period | integer | Simulated time period |
| trust | numeric | Belief that others and institutions will act reliably |
| legitimacy | numeric | Perceived rightfulness and fairness of contribution system |
| enforcement | numeric | Strength of formal enforcement |
| norm_strength | numeric | Strength of cooperative norms |
| coordination | numeric | Alignment of contribution expectations |
| monitoring | numeric | Observability of contribution and defection |
| selective_incentives | numeric | Contribution-linked incentives or rewards |
| scale_complexity | numeric | Difficulty associated with scale and heterogeneity |
| perceived_fairness | numeric | Perceived fairness of burdens and benefits |
| capture_risk | numeric | Risk of narrow-interest capture |
| distributional_attention | numeric | Attention to unequal burden and affected communities |
| contribution_rate | numeric | Synthetic contribution-rate measure |
| provision_quality | numeric | Synthetic public goods provision-quality score |
| high_provision | binary | Strong provision threshold indicator |
| fragile_public_good | binary | High provision with low legitimacy |
| high_burden_risk | binary | High provision with weak distributional attention |
| free_ride_opportunity | numeric | Perceived opportunity to benefit without contributing |
| institutional_competence | numeric | Ability to convert contribution into provision |
