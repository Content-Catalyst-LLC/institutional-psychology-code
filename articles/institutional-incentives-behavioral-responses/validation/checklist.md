# Institutional Incentives Validation Checklist

Before adapting this scaffold to a real case, review:

- [ ] Is the incentive system clearly defined?
- [ ] Are rewards, penalties, metrics, rankings, and evaluation rules documented?
- [ ] Is value alignment assessed?
- [ ] Is fairness assessed?
- [ ] Is legitimacy assessed?
- [ ] Is information quality assessed?
- [ ] Is learning support assessed?
- [ ] Is accountability for incentive consequences assessed?
- [ ] Is metric substitution assessed?
- [ ] Is reporting distortion assessed?
- [ ] Is behavioral burden assessed?
- [ ] Is short-termism assessed?
- [ ] Is status inequality assessed?
- [ ] Is motivation crowding assessed?
- [ ] Are affected communities visible in the analysis?
- [ ] Are hidden labor and burden visible?
- [ ] Are qualitative sources included?
- [ ] Are historical sources included?
- [ ] Are assumptions documented?
- [ ] Is uncertainty documented?
- [ ] Are results framed as interpretive rather than deterministic?
- [ ] Is the analysis free from workplace surveillance or individual assessment uses?
