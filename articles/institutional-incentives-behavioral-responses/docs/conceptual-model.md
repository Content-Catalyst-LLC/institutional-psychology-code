# Conceptual Model

A semi-formal model of incentive effectiveness:

```text
IE = f(VA, FR, IQ, LG, LS, AC, CB, MS, RD, BD, ST, SI, CO)
```

Where:

| Symbol | Meaning |
|---|---|
| IE | Incentive Effectiveness |
| VA | Value Alignment |
| FR | Fairness Perception |
| IQ | Information Quality |
| LG | Legitimacy |
| LS | Learning Support |
| AC | Accountability |
| CB | Cognitive Bias Pressure |
| MS | Metric Substitution |
| RD | Reporting Distortion |
| BD | Behavioral Burden |
| ST | Short-Termism |
| SI | Status Inequality |
| CO | Motivation Crowding |

A simplified incentive-effectiveness score:

```text
IE = β1(VA) + β2(FR) + β3(IQ) + β4(LG) + β5(LS) + β6(AC)
   − β7(CB) − β8(MS) − β9(RD) − β10(BD) − β11(ST) − β12(SI) − β13(CO)
```

A simplified behavioral response model:

```text
P(ChooseAction_i) = σ(α0 + α1V_i + α2F_i + α3N_i + α4I_i + α5L_i
                         − α6B_i − α7M_i)
```

Where:

- `V_i` = perceived reward value
- `F_i` = fairness perception
- `N_i` = normative support
- `I_i` = information quality
- `L_i` = legitimacy of institutional objective
- `B_i` = behavioral burden
- `M_i` = metric-substitution pressure

## Interpretation

- Incentives do not merely move behavior; they define what the institution makes worth doing.
- Incentive effectiveness depends on alignment, fairness, information quality, legitimacy, learning, and accountability.
- High apparent performance can be fragile when metric substitution, burden, reporting distortion, short-termism, status inequality, or motivation crowding are high.
