# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic incentive unit identifier |
| period | integer | Simulated time period |
| value_alignment | numeric | Degree to which incentives reward behavior aligned with institutional purpose |
| fairness | numeric | Perceived fairness, proportionality, and distributional legitimacy of incentives |
| information_quality | numeric | Accuracy, completeness, auditability, and usefulness of incentive-linked information |
| legitimacy | numeric | Perceived rightfulness and credibility of the incentivized objective |
| learning_support | numeric | Degree to which incentives reward correction, feedback, and adaptation |
| accountability | numeric | Review and answerability for incentive consequences |
| bias_pressure | numeric | Cognitive bias pressure affecting interpretation of incentives |
| metric_substitution | numeric | Degree to which measures replace the real institutional objective |
| reporting_distortion | numeric | Pressure to suppress, reframe, delay, or manipulate information |
| behavioral_burden | numeric | Workload, documentation, moral, coordination, psychological, or administrative burden |
| short_termism | numeric | Pressure toward immediate gains over long-term institutional value |
| status_inequality | numeric | Unequal distribution of prestige, recognition, reward, or voice |
| motivation_crowding | numeric | Displacement of intrinsic, civic, professional, or ethical motivation |
| incentive_effectiveness | numeric | Composite synthetic incentive-effectiveness score |
| high_alignment | binary | High incentive-alignment threshold indicator |
| fragile_incentive_system | binary | High incentive effectiveness with low legitimacy |
| high_burden_incentive_system | binary | High incentive effectiveness with high burden and high metric substitution |
