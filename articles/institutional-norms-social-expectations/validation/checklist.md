# Institutional Norms Validation Checklist

Before adapting this scaffold to a real case, review:

- [ ] Is the norm domain clearly defined?
- [ ] Is normative stability separated from justice?
- [ ] Is compliance separated from consent?
- [ ] Is social enforcement separated from suppressive pressure?
- [ ] Is norm repetition assessed?
- [ ] Is expectation convergence assessed?
- [ ] Is internalization assessed?
- [ ] Is social enforcement assessed?
- [ ] Is legitimacy alignment assessed?
- [ ] Is trust reinforcement assessed?
- [ ] Is role clarity assessed?
- [ ] Is learning capacity assessed?
- [ ] Is alternative norm visibility assessed?
- [ ] Is sanction cost assessed?
- [ ] Is suppressive pressure assessed?
- [ ] Is fragmentation pressure assessed?
- [ ] Is unequal normative burden assessed?
- [ ] Is rigidity pressure assessed?
- [ ] Are norms disaggregated across groups?
- [ ] Are historical harms and institutional memory included?
- [ ] Are affected communities visible in the analysis?
- [ ] Are qualitative sources included?
- [ ] Are complaint and appeal pathways reviewed?
- [ ] Is uncertainty documented?
- [ ] Are results framed as interpretive rather than deterministic?
- [ ] Is the analysis free from surveillance, norm scoring, conformity scoring, or individual assessment uses?
