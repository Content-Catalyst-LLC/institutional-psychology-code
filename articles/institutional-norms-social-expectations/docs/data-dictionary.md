# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic norm-system unit identifier |
| period | integer | Simulated norm period |
| norm_repetition | numeric | Visible repetition and routine enactment of a norm |
| expectation_convergence | numeric | Agreement about what others do and what others expect |
| internalization | numeric | Degree to which actors follow the norm without direct monitoring |
| social_enforcement | numeric | Approval, disapproval, inclusion, exclusion, and reputation pressure |
| legitimacy_alignment | numeric | Perceived rightfulness, fairness, and appropriateness of the norm |
| trust_reinforcement | numeric | Expectation that others will follow the norm consistently and in good faith |
| role_clarity | numeric | Clarity of expected conduct within institutional roles |
| learning_capacity | numeric | Ability to revise norms based on feedback, dissent, failure, and repair |
| alternative_norm_visibility | numeric | Visibility of alternative behavior or rival expectation systems |
| sanction_cost | numeric | Cost of deviating from the existing norm |
| suppressive_pressure | numeric | Degree to which enforcement punishes dissent, challenge, or whistleblowing |
| fragmentation_pressure | numeric | Conflict, rival norms, subgroup divergence, or unstable expectation pressure |
| unequal_normative_burden | numeric | Extra work required from some groups to appear normal, credible, or compliant |
| rigidity_pressure | numeric | Resistance to norm change embedded in routines, incentives, or hierarchy |
| normative_stability | numeric | Composite synthetic norm-stability score |
| high_coordination | binary | High-coordination threshold indicator |
| fragile_normative_environment | binary | High apparent stability with weak expectation convergence and weak legitimacy alignment |
| suppressive_norm_environment | binary | Strong social enforcement plus high suppressive pressure and weak learning capacity |
| norm_change_readiness | numeric | Composite synthetic norm-change readiness score |
