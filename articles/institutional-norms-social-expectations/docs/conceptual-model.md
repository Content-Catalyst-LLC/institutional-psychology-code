# Conceptual Model

A semi-formal model of normative institutional stability:

```text
NS = f(NR, EC, IN, SE, LA, TR, RC, LC, CF, SP, UNB)
```

Where:

| Symbol | Meaning |
|---|---|
| NS | Normative Stability |
| NR | Norm Repetition |
| EC | Expectation Convergence |
| IN | Internalization |
| SE | Social Enforcement |
| LA | Legitimacy Alignment |
| TR | Trust Reinforcement |
| RC | Role Clarity |
| LC | Learning Capacity |
| CF | Conflict or Fragmentation Pressure |
| SP | Suppressive Pressure |
| UNB | Unequal Normative Burden |

A simplified norm-stability score:

```text
NS = β1(NR) + β2(EC) + β3(IN) + β4(SE) + β5(LA)
   + β6(TR) + β7(RC) + β8(LC)
   − β9(CF) − β10(UNB) − β11(SP)
```

A norm-change readiness equation:

```text
NCR = θ1(ANV) + θ2(LC) + θ3(LA)
    − θ4(SC) − θ5(RP) − θ6(SP)
```

Where:

- `NCR` = norm-change readiness
- `ANV` = alternative norm visibility
- `SC` = sanction cost
- `RP` = rigidity pressure
- `SP` = suppressive pressure

A threshold model:

```text
Pr(NC = 1) = 1 / (1 + exp(-(θ0 + θ1ANV + θ2LC + θ3LA − θ4SC − θ5RP − θ6SP)))
```

## Interpretation

- Norms are not merely culture.
- Expectations are coordination mechanisms.
- Social enforcement is double-edged.
- Normative stability is not the same as justice.
- Deviance can be harmful, but it can also reveal institutional failure.
- Norm change depends on visible alternatives, lowered sanction cost, legitimacy, and learning capacity.
