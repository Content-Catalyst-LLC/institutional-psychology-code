# Institutions and Human Behavior Validation Checklist

Before adapting this scaffold to a real case, review:

- [ ] Is the institutional domain clearly defined?
- [ ] Is formal structure separated from behavioral reality?
- [ ] Is compliance separated from voluntary alignment?
- [ ] Is institutional durability separated from justice?
- [ ] Is normative stability assessed?
- [ ] Is legitimacy strength assessed?
- [ ] Is incentive alignment assessed?
- [ ] Is information quality assessed?
- [ ] Is memory retention assessed?
- [ ] Is learning capacity assessed?
- [ ] Is trust reinforcement assessed?
- [ ] Is role clarity assessed?
- [ ] Is repair capacity assessed?
- [ ] Is administrative burden assessed?
- [ ] Is opacity pressure assessed?
- [ ] Is historical harm pressure assessed?
- [ ] Is fragmentation pressure assessed?
- [ ] Is institutional behavior disaggregated across groups?
- [ ] Are historical harms and institutional memory included?
- [ ] Are affected communities visible in the analysis?
- [ ] Are qualitative sources included?
- [ ] Are complaint and appeal pathways reviewed?
- [ ] Is uncertainty documented?
- [ ] Are results framed as interpretive rather than deterministic?
- [ ] Is the analysis free from surveillance, institutional scoring, trust scoring, behavior scoring, or individual assessment uses?
