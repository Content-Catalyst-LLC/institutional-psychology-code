# Conceptual Model

A semi-formal model of institutional durability:

```text
ID = f(NS, LG, IA, IF, MR, LC, TR, RC, RP, CF, OP, AB, HH)
```

Where:

| Symbol | Meaning |
|---|---|
| ID | Institutional Durability or Behavioral Reality |
| NS | Normative Stability |
| LG | Legitimacy Strength |
| IA | Incentive Alignment |
| IF | Information-Flow Quality |
| MR | Memory Retention |
| LC | Learning Capacity |
| TR | Trust Reinforcement |
| RC | Role Clarity |
| RP | Repair Capacity |
| CF | Conflict or Fragmentation Pressure |
| OP | Opacity Pressure |
| AB | Administrative Burden |
| HH | Historical Harm Pressure |

A simplified institutional-strength score:

```text
ID = β1(NS) + β2(LG) + β3(IA) + β4(IF) + β5(MR)
   + β6(LC) + β7(TR) + β8(RC) + β9(RP)
   − β10(CF) − β11(OP) − β12(AB) − β13(HH)
```

A behavioral-alignment equation:

```text
BA = δ1(ID) + δ2(LG) + δ3(NS) + δ4(IA) + δ5(TR) + δ6(RC)
   − δ7(CF) − δ8(OP) − δ9(AB)
```

Where:

- `BA` = behavioral alignment
- `ID` = institutional durability or strength
- `CF` = fragmentation pressure
- `OP` = opacity pressure
- `AB` = administrative burden

An institutional-fragility equation:

```text
FR = λ1(CF) + λ2(DT) + λ3(OP) + λ4(AB) + λ5(HH)
   − λ6(LG) − λ7(TR) − λ8(LC) − λ9(RP)
```

Where:

- `FR` = institutional fragility
- `DT` = distrust
- `HH` = historical harm or institutional betrayal memory
- `RP` = repair capacity

## Interpretation

- Institutions are behavioral systems, not only formal rule systems.
- Formal authority is not the same as behavioral legitimacy.
- Compliance is not the same as voluntary alignment.
- Institutional memory can support learning, but it can also preserve bias.
- Institutional durability is not the same as justice.
- Institutional psychology should support accountability, repair, and learning, not manipulation or surveillance.
