# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic institutional-behavior unit identifier |
| period | integer | Simulated institutional period |
| normative_stability | numeric | Stability of shared norms, expectations, routines, and role conduct |
| legitimacy_strength | numeric | Perceived rightfulness, fairness, and acceptability of institutional authority |
| incentive_alignment | numeric | Alignment between incentives and institutionally desired behavior |
| information_quality | numeric | Clarity, accessibility, accuracy, timeliness, and interpretability of institutional information |
| memory_retention | numeric | Capacity to preserve records, precedent, context, and institutional memory |
| learning_capacity | numeric | Capacity to absorb feedback, correct failure, and change institutional behavior |
| trust_reinforcement | numeric | Expectation that institutions and other actors will act consistently and in good faith |
| role_clarity | numeric | Clarity of institutional roles, responsibilities, obligations, and expected conduct |
| repair_capacity | numeric | Capacity to acknowledge, correct, and repair institutional failure |
| administrative_burden | numeric | Proof burden, delay, procedural work, friction, and compliance burden |
| opacity_pressure | numeric | Difficulty understanding or contesting institutional decisions |
| historical_harm_pressure | numeric | Weight of remembered or documented institutional harm, exclusion, or betrayal |
| fragmentation_pressure | numeric | Rival expectations, distrust, subgroup divergence, and narrative conflict |
| institutional_strength | numeric | Composite synthetic institutional-strength score |
| behavioral_alignment | numeric | Composite synthetic behavioral-alignment score |
| high_institutional_alignment | binary | High institutional alignment threshold indicator |
| high_behavioral_alignment | binary | High behavioral alignment threshold indicator |
| fragile_institutional_environment | binary | High apparent institutional strength with weak legitimacy and weak normative stability |
| high_fragmentation_environment | binary | Strong fragmentation and opacity with weak repair capacity |
