# Model Card: Institutional Learning Scaffold

## Model purpose

This scaffold demonstrates how synthetic institutional data can be used to explore learning capacity, feedback quality, memory retention, communication openness, interpretive quality, decision revisability, psychological safety, accountability reach, disconfirming evidence, institutional inertia, signal distortion, memory decay, defensive routines, power protection, and feedback delay.

## Intended users

- researchers
- students
- policy analysts
- public-interest technologists
- institutional scholars
- governance researchers
- organizational researchers
- learning-system researchers

## Out-of-scope users

- employers evaluating workers
- agencies ranking people or communities
- punitive learning-culture systems
- automated decision systems
- surveillance systems

## Inputs

Synthetic values representing learning-system conditions.

## Outputs

- learning capacity score
- high adaptation indicator
- fragile learning indicator
- high inertia learning indicator
- dynamic learning simulation results
- feedback-delay sensitivity analysis
- institutional-inertia review

## Limitations

- synthetic data only
- no real causal validation
- no automatic policy recommendation
- no moral or legal responsibility assignment
- no classification of real actors or communities
- no substitute for legal, historical, qualitative, or institutional expertise

## Ethical risks

- false precision
- treating report production as learning
- ignoring affected communities
- legitimizing symbolic learning systems
- using learning models to manage dissent rather than revise power
