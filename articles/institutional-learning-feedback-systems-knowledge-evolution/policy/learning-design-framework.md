# Institutional Learning Design Framework

## Learning quality criteria

An institutional learning system should be evaluated across multiple criteria:

1. **Feedback quality**  
   Are signals timely, accurate, relevant, interpretable, and broad enough to include affected experience?

2. **Communication openness**  
   Can bad news, weak signals, dissent, and disconfirming evidence travel?

3. **Memory retention**  
   Are lessons stored, contextualized, retrievable, and operationally useful?

4. **Interpretive quality**  
   Are alternative explanations considered and assumptions tested?

5. **Decision revisability**  
   Can policies, routines, metrics, incentives, budgets, and authority structures change?

6. **Psychological safety**  
   Can actors surface error, uncertainty, dissent, and near misses without retaliation?

7. **Accountability reach**  
   Does learning reach actors with authority and responsibility?

8. **Justice and voice**  
   Do marginalized and affected communities shape what counts as evidence and improvement?

9. **Double-loop capacity**  
   Can the institution revise assumptions, not only procedures?

10. **Deutero-learning**  
   Does the institution improve its own ability to learn?

## Outcome typology

| Outcome | Description |
|---|---|
| Information accumulation | Data or reports exist but do not change behavior |
| Single-loop learning | Procedures improve within existing assumptions |
| Double-loop learning | Assumptions, metrics, goals, or governing logics change |
| Deutero-learning | The institution improves how it learns |
| Symbolic learning | Learning language protects reputation without changing practice |
| Fragile learning | Apparent adaptation exists with weak openness or safety |
| High-inertia learning | Apparent learning exists while inertia and distortion remain high |
| Justice-centered learning | Affected voices shape memory, revision, and accountability |
