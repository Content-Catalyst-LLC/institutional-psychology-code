# Institutional Learning: Feedback Systems and Knowledge Evolution

This advanced companion repository supports the article **Institutional Learning: Feedback Systems and Knowledge Evolution** in the Institutional Psychology knowledge series.

The repository provides a research-grade, synthetic-data scaffold for studying how feedback quality, memory retention, communication openness, interpretive quality, decision revisability, psychological safety, accountability reach, disconfirming evidence, institutional inertia, signal distortion, memory decay, defensive routines, power protection, and feedback delay shape institutional learning capacity.

## Research purpose

This repository is designed for:

- synthetic-data research
- institutional learning modeling
- feedback-system simulation
- memory-retention analysis
- communication-openness diagnostics
- psychological-safety and dissent analysis
- decision-revisability assessment
- institutional-inertia review
- signal-distortion analysis
- feedback-delay sensitivity analysis
- power-protection and silence diagnostics
- fragile learning review
- high-inertia learning review
- public administration, organizational strategy, platform governance, regulatory learning, risk and resilience, healthcare safety, education systems, and environmental governance examples
- reproducible multi-language workflows
- teaching and public-interest methods demonstration

It is not designed for automated decision-making about real people, employees, citizens, patients, students, applicants, agencies, firms, communities, or institutions.

## Core research question

When does institutional experience become durable learning that revises assumptions, routines, incentives, memory, and governance — and when do institutions merely collect feedback, publish reports, preserve power, and repeat the same failures?

## Main variables

| Variable | Meaning |
|---|---|
| feedback_quality | Timeliness, accuracy, relevance, coverage, and interpretability of feedback |
| memory_retention | Capacity to preserve, retrieve, contextualize, and reuse institutional knowledge |
| communication_openness | Degree to which evidence, dissent, bad news, and weak signals can travel |
| interpretive_quality | Capacity to analyze feedback, test assumptions, and distinguish signal from noise |
| decision_revisability | Degree to which policies, routines, metrics, budgets, and authority structures can be revised |
| psychological_safety | Safety for surfacing error, dissent, uncertainty, and disconfirming evidence |
| accountability_reach | Degree to which learning reaches actors with authority and responsibility |
| disconfirming_evidence | Evidence that challenges existing assumptions, metrics, or narratives |
| institutional_inertia | Resistance from established routines, budgets, narratives, power, and metrics |
| signal_distortion | Filtering, softening, delay, or manipulation of information as it moves through systems |
| memory_decay | Loss of knowledge through turnover, weak records, inaccessible archives, or fading attention |
| defensive_routines | Practices that protect existing assumptions from challenge |
| power_protection | Protection of authority, reputation, status, or dominant narratives from learning |
| feedback_delay | Time lag between action, consequence, recognition, and revision |
| learning_capacity | Composite synthetic institutional learning-capacity score |
| high_adaptation | Binary threshold indicator for high learning/adaptation capacity |
| fragile_learning | High apparent adaptation with weak communication openness |
| high_inertia_learning | High apparent adaptation with high inertia and signal distortion |

## Responsible-use note

All data is synthetic. The repository is intended for conceptual modeling, public-interest research, education, and transparent methods demonstration.

Do not use this repository for:

- employee screening
- employment selection
- hiring
- promotion
- compensation
- discipline
- termination
- workplace surveillance
- individual performance management
- psychological assessment
- predictive policing
- automated eligibility determination
- coercive public administration
- ranking real communities, agencies, firms, workers, students, or institutions without context
- suppressing dissent under the language of learning, adaptation, culture, safety, improvement, or governance
- extracting testimony from affected communities without changing decisions
- legitimacy laundering for already-decided reform, review, consultation, or learning agendas

Any real-world adaptation must include legal review, ethical review, qualitative evidence, historical analysis, stakeholder participation, burden audits, affected-community voice, and attention to marginalized communities.

## Suggested workflow

1. Read `docs/research-design.md`.
2. Review `ethics/responsible-use-boundaries.md`.
3. Generate synthetic learning data with `python/generate_institutional_learning_data.py`.
4. Run dynamic learning simulation with `python/simulate_learning_dynamics.py`.
5. Run fragile learning diagnostics with `python/fragile_learning_review.py`.
6. Run high-inertia learning review with `python/high_inertia_learning_review.py`.
7. Run feedback-delay sensitivity analysis with `python/feedback_delay_sensitivity.py`.
8. Analyze learning outcomes in R with `r/institutional_learning_modeling.R`.
9. Create SQLite schema with `sql/schema.sql`.
10. Review `validation/validation-plan.md` and `validation/checklist.md`.
11. Use `docs/github-embed-wordpress.html` for the WordPress GitHub block.
