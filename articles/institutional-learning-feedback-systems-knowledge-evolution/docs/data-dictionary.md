# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic institutional unit identifier |
| period | integer | Simulated time period |
| feedback_quality | numeric | Timeliness, accuracy, relevance, coverage, and interpretability of feedback |
| memory_retention | numeric | Capacity to retain, retrieve, contextualize, and reuse knowledge |
| communication_openness | numeric | Degree to which information, dissent, and bad news can travel |
| interpretive_quality | numeric | Capacity to analyze evidence and distinguish signal from noise |
| decision_revisability | numeric | Degree to which rules, routines, metrics, and decisions can be changed |
| psychological_safety | numeric | Safety for surfacing error, dissent, uncertainty, and disconfirming evidence |
| accountability_reach | numeric | Degree to which learning reaches actors with authority and responsibility |
| disconfirming_evidence | numeric | Evidence challenging existing assumptions, categories, metrics, or narratives |
| institutional_inertia | numeric | Resistance from routines, budgets, narratives, power, and metrics |
| signal_distortion | numeric | Filtering, softening, delay, or manipulation of information |
| memory_decay | numeric | Loss of knowledge through turnover, weak records, or inaccessible archives |
| defensive_routines | numeric | Practices that protect existing assumptions from challenge |
| power_protection | numeric | Protection of authority, reputation, status, or dominant narratives |
| feedback_delay | numeric | Lag between action, consequence, recognition, interpretation, and revision |
| learning_capacity | numeric | Composite synthetic institutional learning-capacity score |
| high_adaptation | binary | High learning-capacity threshold indicator |
| fragile_learning | binary | High adaptation with weak communication openness |
| high_inertia_learning | binary | High adaptation with high inertia and signal distortion |
