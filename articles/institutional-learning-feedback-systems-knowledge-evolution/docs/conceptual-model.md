# Conceptual Model

A semi-formal model of institutional learning capacity:

```text
IL = f(FQ, MR, CO, IQ, DR, PS, AR, DE, IN, SD, MD, DF, PP, FD)
```

Where:

| Symbol | Meaning |
|---|---|
| IL | Institutional Learning Capacity |
| FQ | Feedback Quality |
| MR | Memory Retention |
| CO | Communication Openness |
| IQ | Interpretive Quality |
| DR | Decision Revisability |
| PS | Psychological Safety |
| AR | Accountability Reach |
| DE | Disconfirming Evidence |
| IN | Institutional Inertia |
| SD | Signal Distortion |
| MD | Memory Decay |
| DF | Defensive Routines |
| PP | Power Protection |
| FD | Feedback Delay |

A simplified learning-capacity score:

```text
IL = β1(FQ) + β2(MR) + β3(CO) + β4(IQ) + β5(DR) + β6(PS) + β7(AR)
   + β8(DE) − β9(IN) − β10(SD) − β11(MD) − β12(DF) − β13(PP) − β14(FD)
```

A recursive learning equation:

```text
L(t+1) = L(t) + αF(t) + βM(t) + γA(t) − δB(t)
```

Where:

- `F(t)` = feedback quality
- `M(t)` = memory and knowledge-retention quality
- `A(t)` = adaptive willingness or revision capacity
- `B(t)` = barriers to learning, including inertia, distortion, silencing, and defensive routines

## Interpretation

- Institutions do not learn from data alone.
- Learning requires feedback, memory, interpretation, openness, safety, authority, and revision.
- High apparent adaptation can be fragile when inertia, signal distortion, memory decay, defensive routines, feedback delay, or power protection remain high.
