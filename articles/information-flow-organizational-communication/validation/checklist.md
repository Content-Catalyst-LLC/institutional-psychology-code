# Information Flow Validation Checklist

Before adapting this scaffold to a real case, review:

- [ ] Is the communication system clearly defined?
- [ ] Are signal sources documented?
- [ ] Are formal and informal channels mapped?
- [ ] Are escalation pathways documented?
- [ ] Is signal quality assessed?
- [ ] Is communication quality assessed?
- [ ] Is interpretive integration assessed?
- [ ] Is feedback usability assessed?
- [ ] Is memory retention assessed?
- [ ] Is openness assessed?
- [ ] Is trust assessed?
- [ ] Is escalation access assessed?
- [ ] Is community voice assessed?
- [ ] Is digital transparency assessed?
- [ ] Is distortion loss assessed?
- [ ] Is overload assessed?
- [ ] Is siloing assessed?
- [ ] Is suppression pressure assessed?
- [ ] Is metric tunnel vision assessed?
- [ ] Are affected communities visible in the analysis?
- [ ] Are dissent and bad-news channels protected?
- [ ] Are qualitative sources included?
- [ ] Are historical sources included?
- [ ] Are assumptions documented?
- [ ] Is uncertainty documented?
- [ ] Are results framed as interpretive rather than deterministic?
- [ ] Is the analysis free from workplace surveillance or individual assessment uses?
