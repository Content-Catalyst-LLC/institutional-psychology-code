# Conceptual Model

A semi-formal model of institutional information-flow effectiveness:

```text
IF = f(SQ, CQ, II, FU, MR, OP, EA, TR, CV, DT, DL, OL, SL, SP, MT)
```

Where:

| Symbol | Meaning |
|---|---|
| IF | Information Flow Effectiveness |
| SQ | Signal Quality |
| CQ | Communication Quality |
| II | Interpretive Integration |
| FU | Feedback Usability |
| MR | Memory Retention |
| OP | Openness and Psychological Safety |
| EA | Escalation Access |
| TR | Trust |
| CV | Community Voice |
| DT | Digital Transparency |
| DL | Distortion Loss |
| OL | Overload |
| SL | Siloing |
| SP | Suppression Pressure |
| MT | Metric Tunnel Vision |

A simplified information-flow score:

```text
IF = β1(SQ) + β2(CQ) + β3(II) + β4(FU) + β5(MR)
   + β6(OP) + β7(EA) + β8(TR) + β9(CV) + β10(DT)
   − β11(DL) − β12(OL) − β13(SL) − β14(SP) − β15(MT)
```

A recursive information-stock equation:

```text
I(t+1) = I(t) + αS(t) + βC(t) + γF(t) − δD(t)
```

Where:

- `S(t)` = signal generation
- `C(t)` = communication quality
- `F(t)` = feedback integration
- `D(t)` = distortion, delay, overload, or loss

## Interpretation

- Information does not become institutional knowledge just because it exists.
- Communication effectiveness depends on signal quality, transmission, interpretation, openness, memory, trust, and authority access.
- High communication volume can hide fragility when overload, distortion, suppression, false coherence, and metric tunnel vision remain high.
