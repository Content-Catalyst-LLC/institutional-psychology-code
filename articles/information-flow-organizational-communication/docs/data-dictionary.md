# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic institutional communication unit identifier |
| period | integer | Simulated time period |
| signal_quality | numeric | Accuracy, relevance, timeliness, coverage, and contextual value of signals |
| communication_quality | numeric | Reliability, clarity, completeness, handoff quality, and channel fit |
| interpretive_integration | numeric | Capacity to synthesize information across roles, levels, systems, and communities |
| feedback_usability | numeric | Actionability and decision relevance of feedback |
| memory_retention | numeric | Preservation of information with context and future usability |
| openness | numeric | Willingness and safety to surface bad news, weak signals, uncertainty, and dissent |
| escalation_access | numeric | Capacity for relevant signals to reach decision authority |
| trust | numeric | Belief that communication is safe, meaningful, and consequential |
| community_voice | numeric | Inclusion and uptake of affected-community evidence and lived experience |
| digital_transparency | numeric | Data provenance, category visibility, algorithmic traceability, dashboard limits, and technical accountability |
| distortion_loss | numeric | Filtering, softening, reframing, delay, sanitization, politicization, or loss of meaning |
| overload | numeric | Attention burden from excessive dashboards, messages, meetings, alerts, reports, and metrics |
| siloing | numeric | Degree to which knowledge remains trapped in units, professions, platforms, or jurisdictions |
| suppression_pressure | numeric | Fear, incentive, legal, power, or reputational pressure that makes truth costly |
| metric_tunnel_vision | numeric | Overreliance on measurable proxies at the expense of lived and contextual knowledge |
| information_effectiveness | numeric | Composite synthetic information-flow effectiveness score |
| high_integration | binary | High information-integration threshold indicator |
| fragile_communication | binary | High apparent information effectiveness with low openness and high distortion |
| high_overload_system | binary | High apparent information effectiveness with high overload and metric tunnel vision |
