# Compliance and Rule-Following Validation Checklist

Before adapting this scaffold to a real case, review:

- [ ] Is the rule or compliance system clearly defined?
- [ ] Are obligations, deadlines, documentation, enforcement, and appeals documented?
- [ ] Is rule clarity assessed from the user/actor perspective?
- [ ] Is legitimacy assessed?
- [ ] Is fairness assessed?
- [ ] Is trust assessed?
- [ ] Is norm support assessed?
- [ ] Is enforcement credibility assessed?
- [ ] Is compliance burden assessed?
- [ ] Is selective rule application assessed?
- [ ] Is defensive compliance assessed?
- [ ] Are affected communities visible in the analysis?
- [ ] Is noncompliance distinguished from incapacity, ambiguity, exclusion, and justified distrust?
- [ ] Are qualitative sources included?
- [ ] Are historical sources included?
- [ ] Are assumptions documented?
- [ ] Is uncertainty documented?
- [ ] Are results framed as interpretive rather than deterministic?
- [ ] Is the analysis free from workplace surveillance or individual assessment uses?
