# Data Dictionary

| Field | Type | Description |
|---|---:|---|
| unit_id | integer | Synthetic compliance unit identifier |
| agent_id | integer | Synthetic actor identifier |
| period | integer | Simulated time period |
| legitimacy | numeric | Perceived rightfulness, fairness, and authority acceptance |
| fairness | numeric | Perceived fairness of procedures, outcomes, and burden |
| incentive_alignment | numeric | Alignment between incentives and substantive rule purpose |
| norm_support | numeric | Strength of social, professional, or organizational support for rule-following |
| enforcement_credibility | numeric | Consistency, proportionality, and likelihood of enforcement |
| communication_quality | numeric | Clarity, accessibility, and consistency of rule communication |
| cognitive_clarity | numeric | Ease of understanding, remembering, and applying rules |
| trust | numeric | Confidence that institutions and others will behave predictably and consistently |
| adaptive_learning | numeric | Capacity to learn from breakdowns and revise rules or procedures |
| compliance_burden | numeric | Learning, documentation, financial, technological, psychological, and appeal costs |
| selective_rule_application | numeric | Unequal monitoring, sanction, flexibility, or interpretation |
| defensive_compliance | numeric | Blame-avoidant documentation and procedural behavior |
| hypocrisy_visibility | numeric | Visibility of powerful actors avoiding rules or consequences |
| norm_failure | numeric | Breakdown of shared expectation that others are following the rule |
| compliance_quality | numeric | Composite synthetic compliance-quality score |
| high_compliance | binary | High compliance-quality threshold indicator |
| fragile_compliance | binary | High compliance quality with low legitimacy |
| high_burden_compliance | binary | High compliance quality with high burden and selective rule application |
